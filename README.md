Triaxon Software


O Triaxon Software foi desenvolvido para o gerenciar o cadastro de pacientes e médicos em um hospital, além de permitir a pesquisa de informações de ambos.


Funcionalidades:
  Cadastro de Pacientes->
    • Nome
    • CPF
    • Data de Nascimento
    • Telefone
    • Email
    • Endereço

  Cadastro de Médicos:
    • Nome
    • CPF
    • CRM
    • Telefone
    • Email
    • Endereço
    • Data de Admissão
    • Data de Demissão
    • Status

  Pesquisa de Pacientes:
    • Por nome
    • Por datade nascimento
    • Por endereço

  Pesquisa de Médicos:
    • Por nome
    • Por cpf
    • Por crm

Tecnologias utilizadas:
  Frontend->
    ♦ HTML, CSS, JavaScript
    
  Backend->
    ♦ Python, Flask
    
  Banco de Dados->
    ♦ MySQL
